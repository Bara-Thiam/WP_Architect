Fondations solides — 3 points d'ajustement identifiés
WordPress 7.0 et PHP 8.5 témoignent d'une rigueur technique réelle. Mais une extension SEO en retard, trois protections réseau absentes et un score desktop à 45/100 créent des angles morts qui méritent attention — notamment face aux obligations de confidentialité des données de vos dossiers clients.
Point 01
Une stack technique à jour — des choix qui se voient
WordPress Core
7.0 "Armstrong"
Dernière version majeure stable (mai 2026). Peu d'agences immobilières à Dakar ont déjà migré vers cette branche.
Environnement PHP
PHP 8.5
Version active supportée jusqu'à fin 2027. C'est l'environnement optimal pour faire tourner WordPress 7.0 dans les meilleures conditions.
Accès administration
Masqué (404)
Les URLs d'administration standards (/wp-admin, wp-login.php) renvoient une erreur 404. Les robots d'attaque généralistes ne peuvent pas localiser la mire de connexion.
Point 02
Un maillon faible dans la chaîne — l'extension SEO en retard de mise à jour
Yoast SEO
Obsolète
Version détectée : 26.8
Version actuelle : 27.7
Les extensions non mises à jour sont la première cause de compromission sur WordPress — indépendamment de la qualité du CMS.
Risque RGPD lié au Clickjacking
Non protégé
Sans X-Frame-Options, votre site peut être encadré dans une page tierce malveillante pour intercepter les interactions de vos clients lors de la soumission de dossiers.
L'impact concret pour une agence qui collecte des dossiers clients
Vos prospects transmettent des pièces sensibles via vos formulaires (pièces d'identité, bulletins de paie, avis d'imposition). L'absence de HSTS expose ces échanges à une interception sur des réseaux Wi-Fi non sécurisés.
En cas d'incident lié à une extension non patchée, le fait de ne pas avoir appliqué les mises à jour disponibles peut être retenu comme une négligence dans le cadre du RGPD.
Protection manquanteNiveauConséquence
X-Frame-Options
Moyen
Votre site peut être intégré dans une fausse page pour détourner les clics ou intercepter les formulaires de vos clients (Clickjacking).
Strict-Transport-Security
Moyen
La connexion chiffrée HTTPS n'est pas imposée de force au navigateur — les échanges de données peuvent être dégradés vers HTTP sur certains réseaux.
X-Content-Type-Options
Mineur
Le navigateur peut être amené à interpréter des fichiers malveillants comme du code légitime.
Point 03
Un déséquilibre de performances préoccupant entre mobile et desktop
Mobile
71
Desktop
45
Pourquoi un score desktop à 45 est particulièrement dommageable pour l'immobilier
Les fiches de biens immobiliers — avec photos, plans et descriptions détaillées — sont majoritairement consultées sur ordinateur. Un score de 45 signifie que ce parcours clé est fortement dégradé pour vos clients les plus attentifs.
Google PageSpeed intègre les performances desktop dans son algorithme de référencement. Un score en dessous de 50 pénalise directement votre positionnement sur les requêtes de recherche immobilière à Dakar.
Le déséquilibre mobile (71) / desktop (45) indique un problème d'optimisation des ressources lourdes (images non compressées, scripts CSS bloquants) qui se manifeste davantage sur les écrans larges.
